Model in Image

    概要

    Model in Imageの概要、実行時の動作の説明および参照先。

    使用

    Model in Imageイメージとそれに関連付けられたドメインYAMLファイルを作成およびデプロイするステップ。

    補助イメージ

    補助イメージは、ドメイン・モデル・ファイルまたはその他のタイプのファイルを指定するための代替アプローチです。

    モデル・ファイル

    モデル・ファイルの要件、マクロおよびロード順序。

    ランタイムの更新

    実行中のModel in Imageドメイン・イメージおよびモデル・ファイルの更新。

    MII/JRFドメインをPVに移動

    MII/JRFドメインを永続ボリュームに移動します。
