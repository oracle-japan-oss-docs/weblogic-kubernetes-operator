WebLogicのユーザー、グループおよびロールのモデリング

WebLogic Serverには、WebLogicドメイン作成の一環として、ユーザー、グループおよびグローバル・ロールのセットを確立する機能があります。 WebLogicグローバル・ロールは、WebLogicロール・マッパー(XACMLRoleMapperなど)の一部になり、WLSRolesセクションのdomainInfoの下に指定されます。 ユーザーおよびグループは組込みLDAPサーバー(DefaultAuthenticatorなど)の一部になり、Securityセクションのtopologyの下に指定されます。

WebLogicグローバル・ロール

このモデルでは、新しいロールの定義に加えて、既知のWebLogicグローバル・ロール(Admin, Deployer, Monitor、…など)を拡張できるWebLogicロールを定義できます。 既知のWebLogicロールを更新する場合、UpdateModeを{ append | prepend | replace }として指定できます。デフォルトはreplaceです(指定されていない場合)。 また、ウェル・ノウン・ロールを更新する場合、指定されたExpressionはデフォルト式を持つ論理ORになります。 ロールのExpression値は、WebLogicセキュリティ・ロール・マッピング・プロバイダにWebLogic RoleEditorMBeanを使用する場合と同じです。

たとえば、次のWLSRolesセクションでは、ロール名としてTesterを使用して新しいグローバル・ロールを追加しながら、よく知られたAdmin、DeployerおよびMonitorロールを更新します:

domainInfo:
  WLSRoles:
    Admin:
      UpdateMode: append
      Expression: "?weblogic.entitlement.rules.IDCSAppRoleName(AppAdmin,@@PROP:AppName@@)"
    Deployer:
      UpdateMode: replace
      Expression: "?weblogic.entitlement.rules.AdministrativeGroup(@@PROP:Deployers@@)"
    Monitor:
      UpdateMode: prepend
      Expression: "?weblogic.entitlement.rules.AdministrativeGroup(AppMonitors)"
    Tester:
      Expression: "?weblogic.entitlement.rules.IDCSAppRoleName(AppTester,@@PROP:AppName@@)"

Adminロールにはデフォルト式に式が追加され、Deployerロール式によってデフォルトが置き換えられ、Monitorロール式がデフォルト式の先頭に追加され、Testerが指定された式を持つ新しいロールになります。

また、Expression値では、前述の例に示すように、「ツールの作成」の実行時に指定された変数プレースホルダー構文を使用できます。

WebLogicユーザーおよびグループ

このモデルでは、WebLogic組込みLDAPサーバー(たとえば、DefaultAuthenticator)にロードされるユーザーおよびグループのセットを定義できます。 新しいグループを指定し、ユーザーを新しいグループまたは既存のグループ(WebLogic Adminグローバル・ロールにデフォルト設定されるAdministratorsグループなど)のメンバーとして追加できます。 ユーザーおよびグループの詳細は、次の既知の制限事項を参照してください。

ユーザー・パスワードは、プレースホルダーで指定することも、「暗号化ツール」で暗号化することもできます。 グループAppMonitorsを追加し、新しいユーザーを追加し、そのユーザーをグループに配置するSecurityセクションの例は、次のとおりです: 

topology:
  Security:
    Group:
      AppMonitors:
        Description: Application Monitors
    User:
      john:
         Password: welcome1
         GroupMemberOf: [ AppMonitors, Administrators ]
      joe:
         Password: welcome1
         GroupMemberOf: [ AppMonitors ]

既知の制限事項

  • ユーザー、グループおよびロールの処理は、「ドメインの作成ツール」を使用している場合にのみ行われます。
  • WebLogicグローバル・ロールは、WebLogic Serverバージョン12.2.1以上でのみサポートされます。
  • WebLogicグローバル・ロールは、WebLogicセキュリティXACMLロール・マッピング・プロバイダ(XACMLRoleMapperなど)に対してのみ更新されます。
  • ユーザーおよびグループの処理が完了していません。現在、ユーザーをグループに割り当てることはできません。 Securityセクションを使用して作成されたユーザーは、Administratorsグループに自動的に追加され、指定されたグループには追加されません。 この問題のパッチの詳細は、「既知の問題」を参照してください。
