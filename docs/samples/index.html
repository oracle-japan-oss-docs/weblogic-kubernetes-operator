> サンプル
機械翻訳について

サンプル

サンプルでは、一般的なタスクを実行する方法のデモを提供します。 これらのサンプルは、教育目的およびデモ目的でのみ提供されており、本番デプロイメントでの使用や、本番環境の作成に依存することは意図されていません。

    資格証明

    管理サーバー資格証明を含むKubernetesシークレットを作成するためのサンプル。 このシークレットは、WebLogicドメインYAMLファイルの作成に使用できます。

    ストレージ

    WebLogicドメイン・ホームまたはログ・ファイルの永続ストレージとしてドメインYAMLファイルで使用できるPVまたはPVCを作成するためのサンプル。

    データベースの実行

    サンプルまたは基本的なテスト目的に適した一時データベースをKubernetesで実行します。

    ドメイン

    これらのサンプルは、ドメインを操作するための様々な選択肢を示しています。

    REST API

    オペレータ外部REST APIに使用できる自己署名証明書および秘密キーを生成するためのサンプル。

    イングレス

    イングレス・コントローラおよびロード・バランサのサンプル・スクリプト。

    Elastic Stack

    オペレータ・ログ用にElasticsearchおよびKibanaのデプロイメントとサービスを構成するためのオペレータ・サンプル。 WebLogicドメインWebLogicドメインおよびオペレータ・ログにFluentdを使用するためのサンプル。

    Azure Kubernetes Service

    オペレータを使用してAzure Kubernetes ServiceにWLSクラスタを設定するためのサンプル。

    Tanzu Kubernetes Service

    オペレータを使用してTanzu Kubernetes ServiceにWLSクラスタを設定するためのサンプル。
