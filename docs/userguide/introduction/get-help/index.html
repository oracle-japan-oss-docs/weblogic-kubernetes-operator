ヘルプを表示

Oracle Slack

頻繁にチェックされている一般公開のSlackチャネルがあります。ここにオペレータの使用に関する質問や、ご希望の機能や改善についてフィードバックや提案をお寄せください。 また、リクエストに応じて専用のプライベート・チャネルを作成することもできます。 私たちはあなたから便りがあるのが好きです。

パブリック・チャネルに参加するには、「招待を取得するには、このサイトにアクセスしてください」を参照してください。 招待メールには、Slackワークスペースへのアクセス方法の詳細が含まれます。 ログイン後、#operatorに移動して「こんにちは。」

GitHubの問題

すぐにヘルプを得るには、Oracle Slackを使用することを強くお勧めします。 これが不十分な場合は、WebLogic Kubernetes Operatorプロジェクトで「GitHub問題」を送信してください。 すべての問題は公開されています。

Oracleでのサポート

Oracleサポートにアクセスするには、「WebLogic Server My Oracle SupportドキュメントID 2349228.1のKubernetesでの認証」を参照してください。
