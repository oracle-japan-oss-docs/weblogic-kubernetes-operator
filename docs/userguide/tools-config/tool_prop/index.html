ツールのプロパティ・ファイル

ツール・プロパティ・ファイルを使用して、WebLogic Deploy Toolingツールを構成またはチューニングできます。 このプロパティ・ファイルは、<weblogic-deploy>/lib/tool.propertiesとしてインストールされます。 WDTツールをチューニングするために、このファイル内の任意のプロパティの値を変更できます。 別のオプションは、カスタム構成ディレクトリでツール・プロパティを構成することです。 $WDT_CUSTOM_CONFIGディレクトリにtool.propertiesファイルを作成します。

プロパティがファイルから削除された場合、またはプロパティ値の書式が正しくない場合は、WARNINGメッセージがログに記録され、欠落している値や不正な値のかわりに内部のデフォルト値が使用されます。

プロパティ 説明
connect.timeout WLSTがオンラインconnectコマンドの完了を待機するミリ秒数。 値が0の場合、操作はタイムアウトしません。
activate.timeout WLSTが構成変更のアクティブ化の完了を待機するミリ秒数。 値 -1は、操作がタイムアウトしないことを意味します。
deploy.timeout WLSTがアンデプロイメント・プロセスの完了を待機するミリ秒数。 値が0の場合、操作はタイムアウトしません。
redeploy.timeout WLSTが再デプロイ・プロセスの完了を待機するミリ秒数。 値が0の場合、操作はタイムアウトしません。
start.application.timeout WLSTがアプリケーション起動プロセスの完了を待機するミリ秒数。 値が0の場合、操作はタイムアウトしません。
stop.application.timeout WLSTがアプリケーション・プロセスの停止の完了を待機するミリ秒数。 値が0の場合、操作はタイムアウトしません。
set.server.groups.timeout 接続がタイムアウトするまでに、設定されたサーバー・グループの接続を非アクティブにできる時間を指定します。
