WDTツール

    ドメインの作成ツール

    ドメインを作成し、モデルで指定されたすべてのリソースおよびアプリケーションをドメインに移入します。

    ドメインの更新ツール

    既存のドメインを更新し、オフライン・モードまたはオンライン・モードで、モデルに指定されているすべてのリソースおよびアプリケーションをドメインに移入します。

    Deploy Applications Tool

    オフライン・モードまたはオンライン・モードで、既存のドメインにリソースおよびアプリケーションを追加します。

    ドメインの検出ツール

    既存のドメインをイントロスペクトし、ドメインおよびドメインにデプロイされたバイナリのアーカイブ・ファイルを記述するモデル・ファイルを作成します。

    モデルの暗号化ツール

    ユーザー指定のパスフレーズを使用して、モデル(またはその変数ファイル)内のパスワードを暗号化します。

    モデルの検証ツール

    モデルのスタンドアロン検証とモデル用途情報の両方を提供し、ユーザーがモデルを記述または編集できるようにします。

    モデルの比較ツール

    2つのモデル・ファイルを比較します。

    モデルの準備ツール

    WebLogic Kubernetes Operator環境にデプロイするためのモデル・ファイルを準備します。

    ドメイン・リソースの抽出ツール

    WebLogic Kubernetes Operatorで使用するドメイン・リソースYAMLファイルを生成します。

    変数インジェクタ・ツール

    変数を使用してモデルにトークンを追加します。

    モデル・ヘルプ・ツール

    ドメイン・モデルのセクションおよびフォルダに有効なフォルダおよび属性に関する情報を提供します。
