モデルの準備ツール

モデルの準備ツールは、特定のターゲット環境にデプロイするためのWDTモデル・ファイルを準備します。 これらのカスタマイズには次のものが含まれます:

  • 環境と互換性のないセクションのモデルからの削除
  • WDTマクロによる資格証明および属性値の置換
  • 必要なKubernetesシークレットの作成に役立つUNIXシェル・スクリプトの生成
  • 属性値をカスタマイズするための変数プロパティ・ファイルの生成
  • ターゲット環境を構成するための追加の構成ファイルの生成
  • モデル・ファイルの更新

モデルの準備ツールを使用するには、正しい引数を指定してprepareModelシェル・スクリプトを実行します。 有効な引数のリストを表示するには、-helpオプションを指定して(または引数を指定せずに)シェル・スクリプトを実行し、使用方法を確認します。

たとえば、WebLogic Kubernetes Operatorで使用するモデル・ファイルを準備するには、次のように-target wkoでツールを実行します:

$ $WLSDEPLOY_HOME/bin/prepareModel.sh -oracle_home /u01/wls12213 -model_file model1.yaml, model2.yaml -target wko -output_dir $HOME/wko-output

出力ディレクトリには、次のものがあります:

model.yaml
model1.yaml
model2.yaml
create_k8s_secrets.sh
wko_variable.properties

その後、wko_variable.propertiesおよびcreate_k8s_secrets.shをカスタマイズして、環境固有の値を指定できます。

その他のターゲット環境およびオプションの詳細は、「ターゲット環境」を参照してください。
